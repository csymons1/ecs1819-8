What is it?

Scrollspy is used to change the navigation link in a website based on the section you have scrolled to. For example, if your website is broken into pages, you may use dropdown menus at the top as easy navigation.
However, if these sections are all on the same page, Scrollspy is used to show you which section youâ€™re on as you read through (switching the highlighted dropdown).
